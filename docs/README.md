# TCH Music Generation Cloud Platform

## Description
This project aims to provide a comprehensive music generation platform.

## Structure
- backend: Contains backend services
- frontend: Contains frontend application and hooks
- database: Contains database models and migrations
- config: Configuration files for the application
- docker: Docker files for containerization
- k8s: Kubernetes manifests for deployment
- docs: Documentation for the project

## Getting Started

1. Clone the repository
2. Install dependencies
3. Run the application
